# Dootee 🎬

**Dootee** is a simple web application designed to help users in Thailand find which streaming platform is hosting their favorite movies. 

## Features
- Search and filter movies by streaming platforms (Netflix, Disney+, Amazon Prime, Apple TV+).
- Clean and responsive UI.
- Data managed via CSV using PapaParse for easy updating.

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
- [PapaParse](https://www.papaparse.com/) for CSV data handling.

## Deployment
Hosted on [GitHub Pages](https://nspttx-ops.github.io/dootee/).
